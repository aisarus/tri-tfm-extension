# Privacy Policy for TRI·TFM v3.0 Evaluator

**Effective Date:** March 2026

## 1. No Data Collection
The TRI·TFM v3.0 Evaluator Chrome Extension operates entirely locally within your browser. We **do not** collect, store, transmit, log, or share any personal data, user queries, LLM responses, analytics, or telemetry of any kind.

## 2. Interaction with Third-Party Sites
The extension requests permissions to run scripts exclusively on `chatgpt.com` and `claude.ai`. This permission is used **solely** to inject our open-source evaluation prompts (lexemes) into the chat input field upon your explicit manual request (clicking the extension button). We do not intercept, read, or monitor your conversations on these platforms.

## 3. Open Source Transparency
The extension is fully open-source. Anyone can audit the entire codebase at any time to verify our strict no-tracking architecture at:
https://github.com/aisarus/tri-tfm-extension

## 4. Contact
If you have any questions regarding privacy or the extension's behavior, please open an issue on our GitHub repository.
